# MacroRePlayer
universal Macro form application that offers Recording, very diverse Editing with the possibility of adding a loop, and of course playback
# Recorder
records in a clear and universal .json file that can be easily overwritten manually or with an editor or used in any other project
# Editor
offers varied editing of the recorded .json file such as drag and drop of individual actions, editing x,y coordinates of the MouseDown event, TODO adding cycles and TODO speeding up the entire macro by, for example, 1.5x
# Player
classic player nothing special just can TODO bind a key to start/stop a macro
